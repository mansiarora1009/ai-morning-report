## Top 3 Things Worth Paying Attention To

### 1. The harness is the product, not the model
**What it is:** Multiple independent signals converged on this in the last 24 hours. AINews reported that changing prompts and middleware in an agent harness moved gpt-5.2-codex from 52.8% → 66.5% on Terminal-Bench 2.0, and improved gpt-5.3-codex by 20% on tau2-bench — with no model change. Cursor launched a full SDK so developers can build agents on Cursor's own runtime, harness, and models from CI/CD pipelines. Claude Code shipped an "auto mode" for safer permission skipping. The common thread: model quality is no longer the primary differentiator.

**Why it matters:** People in tech are still choosing AI tools by asking "which model is best?" The actual leverage is in how you structure the context pipeline — how repo state is fetched, ranked, compressed into the prompt. Lock-in is coming from harnesses, not weights.

**Reel angle:** "You've been optimizing the wrong thing. The model is a commodity — your harness is the moat."

**What it is:** A developer (@theo) pushed 15 Copilot messages and burned an estimated $221 worth of inference tokens against a $40/month subscription, including one message with 60M+ tokens. This isn't an edge case — it's what happens when any long-running coding agent job hits a platform built on chat-turn economics. Codex UI has started showing usage limit visualizers; cache hit anxiety is a new real phenomenon.

**Why it matters:** Every AI subscription your audience uses — Copilot, Cursor, even Claude.ai — was priced assuming short chat turns. Agentic use breaks that math. Expect usage caps, tier restructuring, or consumption-based pricing to arrive at all of these products within months.

### 3. The METR time-horizon metric explains what "agentic AI" actually means
**What it is:** Import AI published a major essay (Issue #455) arguing 60%+ probability of automated AI R&D by end-2028. The most useful thing in it for a general audience is the METR time-horizon plot: how long a task a skilled human could complete, at the 50% reliability threshold for AI. The numbers: GPT-3.5 (2022) → 30 seconds. GPT-4 (2023) → 4 minutes. o1 (2024) → 40 minutes. GPT-5.2 High (2025) → 6 hours. Opus 4.6 (2026) → 12 hours. METR expects ~100-hour tasks by end of 2026.

**Why it matters:** This is the single clearest frame for explaining to non-researchers what the agentic shift means. "AI can handle 12-hour tasks autonomously" is a concrete, useful sentence. It also explains why coding agents and SWE-bench (now at 93.9% with Claude Mythos Preview, up from ~2% in 2023) are suddenly everywhere.

**Reel angle:** "What does 'agentic AI' actually mean? Here's the one number that explains the whole shift."
