## Top 3 Things Worth Paying Attention To

### 1. Codex is OpenAI's Everything App Now
**What it is:** OpenAI shipped a major Codex expansion alongside GPT-5.5: browser control, Google Sheets/Slides/Docs/PDF reading, OS-wide dictation, and a "guardian agent" auto-review mode that reduces human approval steps on long runs. Ethan Mollick (One Useful Thing) used Codex + GPT-5.5 to produce a near-PhD-quality paper from raw data files in four prompts, and a fully illustrated 101-page TTRPG rulebook in one. OpenAI also officially opened the Codex subscription API to third-party tools — the same integration mechanism Anthropic had blocked for OpenClaw.

**Why it matters to the audience:** Codex is no longer just a code editor in a browser. It's OpenAI's answer to Claude Code, but with browser control, spreadsheets, and presentations baked in. The practical shift: you can now give it a folder of data, a vague goal, and walk away. The "guardian agent" framing is also worth watching — it signals that the future of agentic work isn't more human approvals, it's AI reviewing AI.

### 2. DeepSeek V4 Drops Open-Source — The Efficiency Story Is the Headline
**What it is:** DeepSeek released DeepSeek-V4 Pro (1.6T total params, 49B active, MoE) and V4-Flash (284B/13B active), both MIT licensed with 1M token context. Benchmarks put it at roughly GPT-5.4 / Gemini 3.1 / Opus 4.6 level — solid, not the benchmark leader. The technical story: two new attention mechanisms (CSA and HCA) mean at 1M tokens, V4 uses only 27% of the compute and 10% of the KV cache memory of V3.2. V4-Flash API pricing: $0.14/$0.28 per 1M input/output tokens — the cheapest frontier-class API available right now. Separately notable: both models run on Huawei Ascend chips, a deliberate move away from NVIDIA/CUDA.

**Why it matters to the audience:** The open-weights 1M-context story is now real and cheap. V4-Flash at $0.14/1M input is an order of magnitude cheaper than most alternatives at this quality level. For anyone building agents or doing long-document work, this is a direct "replace your current model" moment. The Huawei Ascend compatibility is a geopolitical signal worth understanding: China is actively building a parallel AI stack that doesn't depend on US chips.

**Format:** Explainer

**Reel angle:** "DeepSeek V4 just dropped open-source with 1M context — skip the benchmarks, the efficiency numbers are what actually matter"

### 3. Voice Dictation Is Finally Fast Enough to Replace Typing
**What it is:** This Week in AI Club ran a head-to-head comparison of every major voice dictation tool: MacWhisper (free, Mac-only, best for just getting started), Superwhisper (most complete — local models, custom vocabulary, tone modes, multi-platform), Wispr Flow (smoothest UX, but $12/month subscription with word limits). All run local Whisper models, meaning no internet required, no cloud uploads.

**Why it matters to the audience:** This isn't "voice dictation kind of works." The reviewer's conclusion: "I can't switch back to just typing." Local models mean privacy. Custom vocabulary means it handles AI jargon. Tone modes mean it can write in your voice for different contexts. This is a workflow upgrade that pays off every single day.

**Format:** First Step

**Reel angle:** "I switched to voice dictation and now I type less than half as much — here's the exact tool to start with"

## Quick Scan

- **Claude Code regression fixed**: Anthropic acknowledged quality slipped in recent versions and patched it in v2.1.116+ — if you've noticed Claude Code getting worse at edits or completions, update now
- **GPT-5.5 "intelligence per dollar"**: GPT-5.5 medium now matches Opus 4.7 max quality at ~¼ the API cost ($1,200 vs $4,800 per evaluation run per Artificial Analysis); frontier intelligence is getting cheaper fast
- **Simon Willison built llm-openai-via-codex**: Three commands and you can run GPT-5.5 from your terminal using your existing ChatGPT subscription — `llm install llm-openai-via-codex && llm -m openai-codex/gpt-5.5 "your prompt"`
- **Agents are systems problems**: Multiple practitioners noted that production agent failures trace to harnesses/evals/memory, not the base model — the Claude Code regression post-mortem was viral specifically because it proved the point
- **Sakana AI launched Fugu**: A multi-agent orchestration API that dynamically routes to different frontier models and claims SOTA on several agentic benchmarks — worth watching as an alternative to hand-rolling orchestration
