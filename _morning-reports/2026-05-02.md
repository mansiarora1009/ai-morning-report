### 1. DeepSeek V4 Pro: the first open-weight model that feels like Claude Code for agentic coding
**What it is:** DeepSeek V4 Pro (1.6T total / 49B active MoE, MIT license, 1M context) is being described by developers as "the first open-weight model that genuinely feels comparable to Codex or Claude Code for multi-turn agentic coding." If you can run frontier-quality multi-step coding loops on Fireworks for a fraction of API cost, the "pay for Claude Code" calculus changes. The model is MIT-licensed and already in OpenRouter.

### 3. The OpenAI–Microsoft AGI clause is officially dead
**What it is:** The longstanding clause that would have voided Microsoft's IP rights to OpenAI technology upon AGI being "declared" has been quietly removed. The new deal (announced April 27) gives Microsoft a non-exclusive license through 2032 "independent of OpenAI's technology progress." Simon Willison tracked the full history: the clause went from "whenever OpenAI achieves AGI" → defined as $100B profit threshold → verified by an independent expert panel → now simply gone.

**Why it matters:** This is one of the clearest signals of what "AGI" actually means in practice to the companies building it — a contractual lever, not a capability threshold. For your audience, this is a useful Explainer on why AGI declarations are fundamentally a business/legal question, not a technical one.
